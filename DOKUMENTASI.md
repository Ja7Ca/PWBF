### Cara Menjalankan Website

## Backend

# Set Up Backend

1. Masuk terimal cmd
2. Masuk direktori server
3. Jalankan 'python -m venv .venv' guna membuat virtual env
4. Jalankan '.venv/Scripts/activate' guna mengaktifkan virtual env
5. Jalankan 'pip install pip --upgrade' guna mengupgrade pip
6. Jalankan 'pip install -r requirements.txt' menginstall module yang ada didalam requirements.txt

# Run Server Backend

1. Masuk terimal cmd
2. Masuk direktori server
3. Jalankan 'python manage.py runserver'

## Frontend

# Set Up Frontend

1. Masuk terminal cmd
2. Masuk direktori website
3. Jalankan 'npm i' guna menginstall semua module yang ada

# Run Server Frontend

1. Masuk terminal cmd
2. Masuk direktori website
3. Ketikkan 'npm start'
4. Tunggu proses, dan website akan muncul otomatis di browser
